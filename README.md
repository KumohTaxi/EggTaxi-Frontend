설치 및 실행 시 kakao map key 공유에 제한이 있어 제대로 된 서비스 확인이 어렵습니다. 배포된 주소를 통해 확인 부탁드립니다.

### 설치
```
$ git clone https://github.com/GHWooo/LIKIT-Frontend.git
$ cd LIKIT-Frontend
```

### 실행
```
$ npm install
$ npm start 
```
